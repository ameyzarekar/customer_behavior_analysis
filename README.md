# customer_behavior_analysis
Data analytics project simulating a corporate workflow. Performed data cleaning using Python, advanced business analysis with MYSQL, and developed interactive dashboards in Power BI. Demonstrates expertise in data modeling, feature engineering, and delivering stakeholder-ready insights on retail customer behavior.

🔍 Overview

This project analyzes **customer shopping behavior** to uncover patterns in purchasing, preferences, and revenue trends. Using **Python, SQL, and Power BI**, the workflow focuses on transforming raw data into meaningful insights for better business decision-making.
📁 Dataset

The dataset is based on **customer shopping behavior analysis**, capturing how customers interact with products and make purchasing decisions.

**Key Attributes:**

* Customer demographics
* Purchase history
* Product categories
* Spending patterns
* Transaction frequency

📌 *Dataset used for portfolio and learning purposes.*
